# joyban 프로젝트 문서

## 프로젝트 개요

개인포트폴리오 및 블로깅 기능이 있는 웹 애플리케이션. 사용자는 컨텐츠(이미지와 텍스트, 첨부파일)을 등록하고 수정, 삭제할 수 있으며 간단한 자기 소개와 소셜 링크를 제공합니다.

**참조UI 이미지**: reference\Desktop.png / reference\Mobile.png

## 주요 기능

- **메인 인덱스 페이지**:
  - **레이아웃**: 참조 이미지 참조.
  - **상단 히어로 섹션**:사이트 제목, 소셜링크, 어드민로그인.
  - **profile 섹션**: 작성된 텍스트 및 이미지.
  - **AIland 섹션**: 클릭하면 별도의 블로그 형식의 동적 페이지로 이동. 게시판 기능이 있는 페이지 생성필요. 네이버 블로그 스타일의 단순한 버전.
  - **works 섹션**: AIland 섹션과 동일.
  - **vision 섹션**: AIland 섹션과 동일.
  - **job 섹션**: AIland 섹션과 동일.
- **컨텐츠 게시 및 관리**:
  - 로그인한 관리자만 등록, 수정, 삭제 가능.
  - **다중 파일 업로드**: 한 번에 여러 장의 사진을 선택하여 일괄 등록 가능.
  - 이미지 파일, 동영상 링크, 사이트 링크 지원.

## 기술 스택

- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript (ES6+).
- **백엔드**: PHP 8.2 (RESTful API 방식).
- **데이터베이스**: MariaDB 10.x (MySQL, PHP PDO 사용).
- **서버**: Cafe24 호스팅 (Linux/Apache).

## 🛠️ 개발 진행 상황 (2026-01-09 업데이트)

### 1. 프로젝트 구조 수립
```text
joyban/
├── api/                    # 백엔드 API (PHP)
│   └── posts/stats.php     # [New] 카테고리별 게시글 통계 API
├── css/                    # style.css (반응형 디자인 적용)
├── js/                     # app.js (프론트엔드 로직 - UIManager 추가)
├── sql/                    # 데이터베이스 관련 (schema.sql)
├── uploads/                # 사용자가 업로드한 파일 저장소
├── index.html              # 메인 페이지 (통계 표시 추가)
├── board.html              # 게시글 목록 페이지 (AIland, Works 등 공용)
├── view.html               # 게시글 상세 페이지 (미디어 타입별 표시 개선)
├── admin_write.html        # 관리자 글쓰기 페이지 (커스텀 알림 적용)
└── GEMINI.md               # 프로젝트 문서 및 진행 가이드
```

### 2. UI/UX 및 기능 개선 (Phase 5)
- **알림 UI 시스템 고도화**:
    - **커스텀 Toast**: 성공/실패 메시지를 우측 상단 슬라이드 알림으로 대체하여 시각적 피드백 강화.
    - **커스텀 Modal**: `alert`, `confirm`, `prompt` 등 브라우저 기본 팝업을 모달 UI(`UIManager` 클래스)로 전면 교체하여 디자인 일관성 확보.
- **첨부파일 처리 고도화**:
    - **이미지**: 썸네일 및 상세 본문에 정상 출력되도록 경로 로직 수정.
    - **동영상**: 본문 내 `<video>` 태그로 자동 변환하여 재생 지원.
    - **기타 파일**: PDF 등은 본문 하단에 별도 다운로드 링크 제공.
- **네비게이션 및 리다이렉트 최적화**:
    - 글 작성/수정 후 메인이 아닌 해당 카테고리 목록으로 이동.
    - 게시글 삭제 후 해당 카테고리 목록으로 정확히 복귀.
    - `view.html`의 '목록으로' 버튼이 현재 카테고리를 유지하도록 수정.
- **메인 페이지 통계**:
    - 각 섹션(AIland, Works 등) 카드에 현재 등록된 게시글 수 실시간 표시 (`stats.php` 연동).

### 3. 버그 수정 및 안정화
- **board.html**:
    - 헤더에 관리자 로그인 버튼 누락으로 인한 권한 체크 오류 수정.
    - 초기 로딩 시 더미 데이터가 노출되지 않도록 구조 변경.
    - 페이지네이션이 실제 데이터 수에 맞춰 동적으로 생성되도록 수정.
- **view.html**:
    - 관리자 컨트롤 JS 에러 수정.
    - 첨부파일 미디어 래퍼(`content-media-wrapper`) 구조 개선.
